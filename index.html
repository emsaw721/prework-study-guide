Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The meta element is important for search terms when trying to get more traffic to your website.
  • The body element represents the visible content shown to the user.
  • There exist unordered lists (bullets), ordered lists, and others.
  • There are up to six types of headings, with 1 being the largest and 6 the smallest.

CSS

  • A margin indicates how much space we want around the outside of an element. The "wall" when haning a picture frame.
  • A padding indicates how much space we want around the content inside an element. The "mat" in a picture frame.
  • The border indicates the parameters of the element. The "picture frame".
  • An asterisk declares ALL elements on the page will have style properties margin and padding as set.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.
  • git branch -d branch name: deletes local branch that has been pushed and merged with remote branch.
  • git branch -D branch name: deletes local branch, even if not pushed or merged.
  • git push remote location --delete branch name OR git push remote location : branch name: deletes branch remotely.
